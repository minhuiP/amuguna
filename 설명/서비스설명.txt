I want to create a website using python, flask, and html where a user can access a SQLite3 database 'amuguna.db' 
and a table 'amuguna_food' with rows including 'food_name', 'tag1', 'tag2', 'tag3', and 'tag4'. 
When the user accesses the website, they will see a randomly selected food_name on the screen, along with 'Select', 'gunang', and 'tag' buttons. 
In the tag button, one of tag1, tag2, tag3, and tag4 of food_name is randomly shown.
If the user clicks the 'Select' button, they will be directed to the 'chuka.html' page. 
If the user clicks the 'gunang' button, a new randomly selected food_name will appear, but it will not include any food_names that contain the tags that have been clicked previously. 
If the user clicks the 'tag' button, a new randomly selected food_name will appear, 
but it will not include the 'tag' that was clicked or any food_names that contain the tags that have been clicked previously. 
This process will continue until all food_names have been excluded, at which point the screen will move to 'donteat.html'.

 Total number of food_name - 'Number of food_name excluded because tag is shown' - 'Number of food_name shown' = 0.
The screen moves to 'donteat.html'.
And food names and tags must not be duplicated.
